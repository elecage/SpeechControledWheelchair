# 보드 출력 Pinout

| pin |   name   |      discription       |
|-----|----------|------------------------|
|  1  |    Vin   | Power supply           |
|  2  |    GND   | Ground                 |
|  3  | JOY_X_IN | Joystick X axis input  | 
|  4  | JOY_Y_IN | Joystick Y axis input  |
|  5  | JOY_VCC  | Joystick power input   |
|  6  | FSR+     | FSR sensor +           |
|  7  | FSR-     | FSR sensor -           |
|  8  | GND      | Ground                 |
|  9  | JOY_X_OUT| Joystick X axis output |
| 10  | JOY_Y_OUT| Joystick Y axis output |
| 11  | GND      | Ground                 |
